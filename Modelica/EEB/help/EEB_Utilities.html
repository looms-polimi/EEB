EEB.Utilities.HPvector_VectOfHPs

HeatPortVector from vector of HeatPort

Information

This model creates a heat port vector from n heat port.

It is used in WaterPipeExchanging_Nvols.

Parameters

Integern2No. of elements

Connectors

Modelica definition

model HPvector_VectOfHPs "HeatPortVector from vector of HeatPort" parameter Integer n = 2 "No. of elements"; Interfaces.Thermal.HeatPortVec vecHP(n = n); Interfaces.Thermal.HeatPort vecOfHPs[n]; equation for i in 1:n loop vecOfHPs[i].T = vecHP.T[i]; vecOfHPs[i].Q_flow + vecHP.Q_flow[i] = 0; end for; end HPvector_VectOfHPs;
